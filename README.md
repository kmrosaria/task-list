# How to run

## Server

Go to server folder using.

```bash
cd server
```

Install modules using: 

```bash
yarn install
```

Run Server: 

```bash
yarn start
```



## Client

Go to client folder using.

```bash
cd client
```

Install modules using: 

```bash
yarn install
```

Run Client: 

```bash
yarn start
```

Open http://localhost:3000 to view it in your browser.
